---
title: 'Create Doctor'
api: 'POST /doctor/create'
---

Create a new doctor account in the system.

## Request Body

<ParamField body="crm" type="string" required>
  Doctor's CRM (Medical Registration) number. Must be unique and up to 9 characters.
</ParamField>

<ParamField body="email" type="string" required>
  Doctor's email address. Must be unique.
</ParamField>

<ParamField body="password" type="string" required>
  Password for the doctor's account.
</ParamField>

<ParamField body="name" type="string" required>
  Doctor's full name.
</ParamField>

<ParamField body="specialty" type="string" required>
  Doctor's medical specialty (e.g., Cardiology, Pediatrics, Dermatology).
</ParamField>

<ParamField body="telephone" type="string" required>
  Doctor's contact telephone number.
</ParamField>

<ParamField body="consultationValue" type="number" required>
  Price for a consultation with this doctor (decimal value).
</ParamField>

## Response

<ResponseField name="crm" type="string">
  Doctor's CRM number.
</ResponseField>

<ResponseField name="email" type="string">
  Doctor's email address.
</ResponseField>

<ResponseField name="password" type="string">
  Doctor's password.
</ResponseField>

<ResponseField name="name" type="string">
  Doctor's full name.
</ResponseField>

<ResponseField name="specialty" type="string">
  Doctor's medical specialty.
</ResponseField>

<ResponseField name="telephone" type="string">
  Doctor's telephone number.
</ResponseField>

<ResponseField name="consultationValue" type="number">
  Consultation price.
</ResponseField>

## Example Request

```bash
curl -X POST http://localhost:8080/doctor/create \
  -H "Content-Type: application/json" \
  -d '{
    "crm": "123456789",
    "email": "dr.silva@hospital.com",
    "password": "securePassword123",
    "name": "Dr. João Silva",
    "specialty": "Cardiology",
    "telephone": "+5534999887766",
    "consultationValue": 250.00
  }'
```

## Example Response

```json
{
  "crm": "123456789",
  "email": "dr.silva@hospital.com",
  "password": "securePassword123",
  "name": "Dr. João Silva",
  "specialty": "Cardiology",
  "telephone": "+5534999887766",
  "consultationValue": 250.00
}
```

## Error Responses

<ResponseField name="400 Bad Request" type="error">
  Invalid request body or missing required fields.
</ResponseField>

<ResponseField name="409 Conflict" type="error">
  Doctor with this CRM or email already exists.
</ResponseField>
