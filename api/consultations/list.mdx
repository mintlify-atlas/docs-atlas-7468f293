---
title: 'List All Consultations'
api: 'GET /consultations/all'
description: 'Retrieve a list of all consultations in the system'
---

Returns a complete list of all consultations scheduled in the system, including past, present, and future appointments. Each consultation includes full patient and doctor information.

## Response

Returns an array of consultation objects.

<ResponseField name="consultations" type="array">
  Array of consultation objects
  
  <Expandable title="Consultation object">
    <ResponseField name="consultationId" type="string">
      UUID of the consultation
    </ResponseField>
    
    <ResponseField name="dateTime" type="string">
      Scheduled date and time in ISO 8601 format
    </ResponseField>
    
    <ResponseField name="duracaoMinutos" type="integer">
      Duration of the consultation in minutes
    </ResponseField>
    
    <ResponseField name="patient" type="object">
      Patient information including cpf, name, email, dateOfBirth, address, and medicalHistory
    </ResponseField>
    
    <ResponseField name="doctor" type="object">
      Doctor information including crm, name, email, specialty, telephone, and consultationValue
    </ResponseField>
    
    <ResponseField name="isUrgent" type="boolean">
      Whether this is an urgent consultation
    </ResponseField>
    
    <ResponseField name="observation" type="string">
      Additional notes or observations
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl https://api.medagenda.com/consultations/all
```

```javascript JavaScript
const response = await fetch('https://api.medagenda.com/consultations/all');
const consultations = await response.json();

console.log(`Total consultations: ${consultations.length}`);
```

```python Python
import requests

response = requests.get('https://api.medagenda.com/consultations/all')
consultations = response.json()

print(f'Total consultations: {len(consultations)}')
```

</RequestExample>
